1.3.1 — reliability and performance:
• Bounded TCP dial timeout fixes occasional connection hangs on DIRECT-routed traffic.
• Cleaner TCP lifecycle when the network path changes (Wi-Fi ⇄ cellular).
• Self-restart on memory pressure keeps the tunnel stable under heavy load.
• Lazy GeoIP/ASN database download — faster first launch, smaller install.
• Subscription updates send the clash.meta User-Agent for better provider compatibility.
• Encrypted Client Hello (ECH) support via boring-tls.
• Minimum supported OS lowered to iOS 17 (iPhone and iPad).
